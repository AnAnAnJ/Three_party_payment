
## 环境搭建

1. 配置venv
   > PyCharm -> Preferences -> Project -> Python Interpreter -> setting按钮 -> add
2. 安装依赖 
   > pip install -r requirements.txt

## 框架说明：

本框架可实现接口的自动化测试 

### 执行流程：

自动化测试用例编写=>自动化执行测试用例=>生成报告=>push飞书

### 框架目录： 
``` 
lib目录：  工具
test_demo目录： 自动化测试用例存放目录 
func目录：  配置文件目录
util目录：   日志
report目录：测试报告存放目录
```
